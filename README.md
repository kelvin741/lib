# Library Management System

A full-stack web application for managing library books and user accounts.

## Project Structure

```
├── frontend/        # Frontend UI (HTML, CSS, JavaScript)
├── backend/         # Backend API (Node.js)
├── .gitignore
└── README.md
```

## Features

- User authentication (Login/Register)
- Role-based access (Student/Admin)
- Book management
- Book search functionality
- Borrow/Return books
- Admin panel for book management

## Getting Started

### Prerequisites
- Node.js installed
- npm or yarn package manager

### Installation

1. **Backend Setup**
```bash
cd backend
npm install
npm start
```

2. **Frontend Setup**
```bash
cd frontend
# Open index.html in a web browser
```

## Usage

- **Students**: Login to search and borrow books
- **Admins**: Add new books and manage the library catalog
